# Fishball - Facebook Auth Project

## Description
A web application that integrates with the Facebook Graph API using OAuth 2.0 to fetch and display user profile data.

## Project Structure
- `index.html`: The main UI.
- `style.css`: Responsive design and custom theme.
- `script.js`: Handles validation, loading states, and redirection.
- `server.js`: Node.js backend for secure OAuth handshake.

## How to Run
1. Run `npm to install express, axios, dotenv, cors, and cookie-parser`.
2. Run the server: `node server.js`.
3. Open `http://localhost:3000` in your browser.
